Frank Tarot is a minimal AI tarot reading experience designed for calm, focused reflection. Choose a spread, ask a question, draw your cards, and receive a reading shaped by card positions, upright or reversed orientation, and the overall energy of the spread.

If an API key is connected, the app generates the reading for you. If not, you can still complete the ritual, copy a ready-made prompt, and continue in ChatGPT.

命运塔罗是一个极简、注重审美的 AI 塔罗体验。你可以选择牌阵、提出问题、抽取卡牌，并根据牌位、正逆位和整体牌阵结构获得一段完整解读。

如果你连接了 API key，应用会直接生成解读；如果没有，也依然可以完成抽牌流程，一键复制整理好的提示词，再去 ChatGPT 继续阅读。

## Why People Use It

- Minimal and aesthetic interface made for a calm tarot session
- AI-assisted readings based on spread structure and card orientation
- Voice guidance for a more ritual-like experience
- A full collection of tarot spreads for love, clarity, reflection, relationship, and more
- English and Simplified Chinese support
- Mobile-friendly and easy to use in the browser

## How It Works

1. Choose a tarot spread.
2. Ask your question, or leave it open for general guidance.
3. Draw and reveal your cards.
4. Read the interpretation generated inside the app.
5. If no API key is available, copy the prompt and continue in ChatGPT.

## Notes

- This project is designed to run in the browser.
- For GitHub Pages, the app is configured under `/frank-tarot/`.
- If you run it locally with your own AI key, create `.env.local` and add `GEMINI_API_KEY=your_api_key_here`.
